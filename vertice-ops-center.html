🔥 Conectar Firebase

Para guardar tus datos en la nube (accesible desde cualquier PC), conecta un proyecto gratuito de Firebase. Solo toma 3 minutos.

Paso 1 — Crear proyecto Firebase (gratis)
Ve a console.firebase.google.com → "Crear proyecto" → ponle nombre (ej: vertice-crm) → desactiva Analytics → Crear.
Paso 2 — Agregar app Web
En el proyecto: icono </> → nombre la app → "Registrar app". Copia los datos de firebaseConfig que aparecen.
Paso 3 — Habilitar Firestore
En el menú izquierdo: Compilación → Firestore Database → "Crear base de datos" → modo Prueba (30 días gratis, luego ajusta reglas) → región us-central1.
Paso 4 — Habilitar Authentication (para Gmail)
Compilación → Authentication → "Comenzar" → pestaña "Sign-in method" → habilita Google → guarda. Agrega tu dominio: Dominios autorizados → Agregar dominio (si usas servidor local).
